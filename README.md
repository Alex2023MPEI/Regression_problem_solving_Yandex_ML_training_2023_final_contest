# Regression_problem_solving_Yandex_ML_training_2023_final_contest
Данные (800 открытых и 200 закрытых образцов), код и результаты для задачи регрессии из финального задания тренировок по машинному обучению от Яндекс за 2023 год.

Задача с тренировок по ML от Яндекс 2023 года [https://yandex.ru/yaintern/training/ml-training-2023]. Финальное соревнование [https://contest.yandex.ru/contest/56809/problems/]. Есть задача регрессии, 7 признаков (все типа float), 800 открытых образцов (со значениями target), 200 закрытых образцов (без значений target). Можно генерировать новые признаки из имеющихся семи (например, признак 7 как сумма признаков 0 и 1 или признак 8 как произведение признаков 3 и 4 и так далее). Задача состоит из двух частей.
В части 1 нужно построить модель без каких-либо ограничений, в контест отправляется json файл с ответами.
В части 2 условие: модель должна быть линейной и иметь не более 15 параметров (14 весов и смещение). В контест нужно отправить py файл со значениями весов, смещения и функцией my_transformation, которая принимает на вход исходную матрицу признаков и возвращает новую матрицу признаков. Таким образом, этот код однозначно определяет преобразование признаков и все 15 параметров модели.

Решение задачи (обеих частей) находится в файле p1.py.
